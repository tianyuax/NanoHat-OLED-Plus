# NanoHat-OLED-Plus
基于NanoPi NEO系列 [NanoHat OLED](http://wiki.friendlyarm.com/wiki/index.php/NanoHat_OLED/zh) 官方开源代码的功能增强。


## 支持设备
* NanoPi NEO
* NanoPi NEO Air
* NanoPi NEO2
* NanoPi NEO Plus2

## 安装说明
移至 ```/root/NanoHatOLED/BakeBit/Software/Python ```
重启设备


## 版本迭代
